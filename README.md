# grocery-retail-sales-analysis
Overview:
Retail analytics project using BigQuery and Tableau to analyze grocery sales records sourced from a public Kaggle dataset (2018 data).

Tools:
BigQuery
Tableau

Data Preparation:
Viewed and cleaned dataset and tables
Created transformation layer using SQL views
Aggregated performance metrics for dashboard optimization
Built KPI summary tables

Metrics:
Total Revenue
Total Units Sold
Total Transactions
Average Order Value

Insights:
Revenue distribution across categories is relatively balanced.
Top-performing products drive significant revenue share.
Revenue trends remain stable across the observed period.
